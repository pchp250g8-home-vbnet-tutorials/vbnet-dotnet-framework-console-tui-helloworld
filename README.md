HelloWorld. VB.Net Tutorials. Platform DotNet FrameWork. TUI. Advanced Features. Console Application.
The program does the following:
  1. Creates a standard terminal size window (80 columns, 25 rows).
  2. Clears the created window.
  3. Displays the first "Hello, world!!!" message in column 10 and row 10.
  4. Displays the second inscription "VB.Net DotNet FrameWork Platform!!!" in column 10 and row 11.
  5. Waiting for a key to be pressed.
  6. Exits.

  4. Выводит на экран вторую надпись "VB.Net DotNet FrameWork Platform!!!" в 10 столбце и 11 строке.
